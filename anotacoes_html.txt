HTML

link normal:
<a href"https">LINK<a/>

link com ancora:
<a href"https..#contents">LINK<a/>

- a ancora define uma localização dentro da pagina para onde o usuário deve ser levado

<img> - imagens dentro da pagina
<img src="foto.png"> - caminho da foto
<img src="foto.png" alt = "imagem"> - alt é uma descrição para a imagem

-Tabelas
<table>

<tr> - linha
<th> - celula de cabeçalho
<td> celula que contém os valores da tabelas

-Listas

<ul> - lista não ordenada
<li> - cada item da lista

<ol> - lista ordenada, os itens da lista serão apresentados com uma notação de números em ordem crescente

-IFrames
<iframe src="htttps"></iframe> utilizado para apresentar um outro website dentro da página website

-Entidade HTML

-alguns caracteres são reservados no HTML, como os símbolos de <e>
-para representar esses caracteres devem ser utilizadas entidades html

-Head
-cabeçalho da pagina html

-Meta
-informações adicionais da página, como por ex palavras chaves
